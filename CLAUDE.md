## UI Review Skill
Use the `/ui-review` skill when asked to audit UI, check design consistency,
or find visual defects. It reviews SwiftUI files in:
- `Covet/Views/` — full-screen views
- `Covet/Components/` — reusable components
- `Covet/Classes/Views/` — legacy views

Design system: Glossier-style (soft whites, 16pt grid, no hardcoded colors).
Entry point: `Covet/CovetApp.swift` (`@main` struct).
P1 screens: Gift Discovery (`GiftFlowView.swift`) and Curated Items for Me.
